# Azure SKU Sizing Tool

A web-based tool to help determine the appropriate Azure VM SKU based on workload requirements.

## Features

- Interactive questionnaire to gather workload requirements
- SKU recommendations based on CPU, memory, storage, and network needs
- Cost-optimized suggestions

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

The app automatically deploys to GitHub Pages when pushing to the `main` branch.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
